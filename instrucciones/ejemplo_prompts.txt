# ============================================================
# CONSEJOS GENERALES
# ============================================================

# 1. CONSISTENCIA DE DESCRIPCIÓN
#    - Usa exactamente la misma descripción base del producto en todos los prompts
#    - Incluye características distintivas: color exacto, forma, tamaño, logo
#    - Especifica detalles visuales que identifiquen el producto

# 2. VARIACIÓN DE CONTEXTO
#    - Cambia el escenario y fondo, NO las características del producto
#    - Describe el contexto de manera clara pero no excesiva
#    - Mantén el producto como el elemento principal

# 3. ILUMINACIÓN Y ATMÓSFERA
#    - Especifica condiciones de iluminación para cada escenario
#    - Describe el ambiente general (energético, relajado, profesional)
#    - Considera cómo la iluminación afecta la apariencia del producto

# 4. ACCIÓN Y MOVIMIENTO
#    - Describe el movimiento deseado del producto o de la cámara
#    - Usa términos específicos: "rota lentamente", "permanece estático", 
#      "se levanta suavemente"
#    - Evita movimientos complejos que puedan generar inconsistencias

# 5. LONGITUD DEL PROMPT
#    - Los prompts más largos y detallados suelen dar mejores resultados
#    - Balancea detalle con claridad
#    - Prioriza la descripción del producto sobre el contexto

# 6. PRUEBAS Y AJUSTES
#    - Experimenta con variaciones del mismo prompt
#    - Compara resultados para identificar qué funciona mejor
#    - Documenta los prompts que generan mejores resultados

# ============================================================
# ESTRUCTURA DE PROMPT RECOMENDADA
# ============================================================

# [Descripción base del producto (SIEMPRE IGUAL)]
# + [Ubicación/contexto del escenario]
# + [Elementos de fondo relevantes]
# + [Condiciones de iluminación]
# + [Atmósfera/ambiente]
# + [Acción o movimiento del producto/cámara]
